# 03. Funcionalidades del Sistema

Este documento describe las capacidades operativas e interactivas disponibles en la plataforma SinVial Ocaña.

## 1. Mapa Interactivo Geo-referenciado
El corazón del sistema es un mapa dinámico basado en Leaflet que permite:
-   **Navegación Fluida:** Zoom, desplazamiento y re-centrado rápido en el casco urbano de Ocaña.
-   **Visualización por Puntos:** Marcadores individuales con iconos semánticos según el tipo de siniestro.
-   **Mapas de Calor (Heatmaps):** Visualización de densidad para identificar zonas de alta siniestralidad (Puntos Negros).
-   **Popups de Detalle:** Acceso inmediato a la ficha técnica de cada siniestro con un solo clic sobre el marcador.

## 2. Paneles de Gestión
-   **Panel de Filtros Colapsable:** Ubicado en el lateral izquierdo, permite segmentar datos por fechas, gravedad, tipo de siniestro y vehículos involucrados.
-   **Panel de Estadísticas (Overlay):** Un panel expansible desde la parte inferior que ofrece un análisis visual mediante KPIs y gráficas de Recharts.
-   **Guía del Mapa (Leyenda):** Widget arrastrable que explica la simbología y colores del sistema.

## 3. Experiencia de Usuario Avanzada
-   **Selector de Temas:** Cambio instantáneo entre modo Claro (Beige Premium) y modo Oscuro (Diseño Original).
-   **Selector de Capas:** Cambio entre mapas base (Satélite, Oscuro, Topográfico, OSM).
-   **Modo Pantalla Completa:** Optimización del espacio de trabajo ocultando el header institucional.
-   **Badges de Víctimas:** Visualización directa del número de víctimas en el icono del mapa mediante un distintivo rojo para siniestros graves.

## 4. Reportes y Exportación
El sistema permite la salida de datos en formatos estándar directamente desde el header:
-   **Exportación CSV:** Descarga del set de datos filtrado actualmente.
-   **Exportación PDF:** Generación de un reporte visual y tabular de los incidentes.
